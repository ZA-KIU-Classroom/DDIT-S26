# Activity: Strategic Options Analysis

## Purpose

Evaluate three structurally distinct strategic options using a consistent criteria matrix and produce a defended recommendation.

## Output
Strategic Options Analysis v1

## Time
40 minutes

## Process

1. Part 1 (10 minutes): Write a clear one-paragraph description of each option. Specific enough that a board member could understand it without additional context.

2. Part 2 (15 minutes): Complete the 3x4 criteria matrix. Each cell: a score (High, Medium, Low) and one sentence of justification referencing evidence from your competitive landscape or AI opportunity map.

3. Part 3 (10 minutes): Write a financial logic paragraph for each option. Describe the mechanism of value creation and the key financial assumption that determines ROI.

4. Part 4 (5 minutes): Write a recommendation paragraph. Lead with the recommendation (Pyramid Principle). State three reasons. State one condition that would change the recommendation.

## Four evaluation criteria

| Criterion | What it measures |
|---|---|
| Strategic fit | How well this option addresses the core problem from your issue tree |
| Financial return | Estimated ROI, payback period, and NPV in base case |
| Implementation feasibility | Whether the org has the capabilities, data, and capital required |
| Competitive defensibility | How defensible the resulting position is against platform entrants and AI-native competitors |

## Common error

Presenting three versions of the same option at different levels of ambition. Three intensities are not three options. Options must be structurally different: different business model logic, different customer focus, or different operating model architecture.
