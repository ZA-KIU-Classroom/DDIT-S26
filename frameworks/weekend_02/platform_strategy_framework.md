# Platform Strategy Framework

## Source
Parker, Van Alstyne, and Choudary. Platform Revolution (2016).

## Core definition

A platform is a business that facilitates interactions between two or more distinct user groups who provide each other with network benefits. The platform does not produce value itself. It creates the conditions for others to produce and exchange value.

## Key concepts

| Concept | Definition |
|---|---|
| Pipeline vs. Platform | Pipeline creates value through a linear process. Platform creates value by facilitating interactions between external participants. |
| Two-sided market | Two distinct user groups whose participation makes the platform more valuable for the other side. |
| Network effects (direct) | More users on one side makes the platform more valuable for same-side users. |
| Network effects (indirect) | More users on one side makes the platform more valuable for other-side users. |
| Data network effects | More users generate more data, improving the AI, attracting more users. |
| Value unit | The minimum unit of interaction between producers and consumers. |
| Chicken-and-egg problem | Both sides must be present simultaneously for the platform to be valuable to either. |

## Strategic options for incumbents

| Option | When it makes sense | Financial logic |
|---|---|---|
| Build a platform | Large loyal user base on one side; data assets; can fund the cold-start problem | High risk, high return |
| Join a platform | Core value is as a producer where a dominant platform controls customer access | Lower risk, lower return |
| Defend against a platform | Regulatory advantages or data assets the platform cannot replicate | Risk reduction investment |
| Acquire a platform | Capital and strategic motivation to control rather than build from scratch | Faster but expensive |

## How to use in this course

1. Classify the client's current business model as pipeline or platform.
2. Identify platform entrants and the value unit they compete over.
3. Assess which network effect type applies and the critical mass threshold.
4. Make a build, join, defend, or acquire recommendation in the Platform Strategy Canvas.
